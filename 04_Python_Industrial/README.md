# 🐍 Fundamentos de Python para Automatización

En esta sección documento mis avances en la lógica de programación utilizando Python. Aunque son ejercicios de base, se enfocan en habilidades críticas para el sector industrial: gestión de datos, control de errores y lógica estructurada.

## 📁 Proyectos incluidos:

### 1. Gestión de Listas Interactiva (`gestion_listas_interactiva.py`)
**Descripción:** Un sistema de menú interactivo para gestionar una base de datos de alumnos (o activos).
- **Habilidades técnicas:** Uso de bucles `while` infinitos con salida controlada, manipulación de listas (`append`, `pop`, `sort`) y creación de funciones propias para limpiar pantalla y navegación.
- **Aplicación industrial:** Base para crear menús de configuración en terminales o gestión de inventarios de repuestos.

### 2. Diccionario de Países UE (`diccionario_paises_ue.py`)
**Descripción:** Consulta dinámica de capitales mediante una base de datos interna.
- **Habilidades técnicas:** Estructuras de datos tipo *Diccionario* (Key-Value), normalización de texto (`.lower()`, `.strip()`) para evitar errores de usuario.
- **Aplicación industrial:** Asociación de IDs de sensores a descripciones técnicas o unidades de medida.

### 3. Cálculo con Control de Errores (`calculo_raices_seguro.py`)
**Descripción:** Calculadora que evita fallos críticos mediante condicionales.
- **Habilidades técnicas:** Manejo de tipos `float` y lógica booleana para prevención de errores matemáticos.
- **Aplicación industrial:** Algoritmos de seguridad que evitan que un proceso se detenga por un cálculo inválido.
